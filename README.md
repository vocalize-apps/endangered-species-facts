# :paw_prints: Endangered Species Facts

An voice app that gives interesting facts about endangered species and promotes awareness about them. Made for Alexa and Google Assistant!

[![Enable This Skill](https://dabuttonfactory.com/button.png?t=Enable+This+Skill&f=Calibri-Bold&ts=24&tc=fff&tshs=1&tshc=000&hp=20&vp=8&c=5&bgt=gradient&bgc=0ff&ebgc=187cd8&shs=1&shc=444&sho=s)](http://bit.ly/ALEXAESF)

## :wave: [Contribute interesting facts here](https://airtable.com/shrKoriHZsn3tz9qU)

## Usage

> "Alexa or Hey Google, ask endangered species facts to tell me a random fact"

> "Alexa or Hey Google, ask endangered species facts to tell me a tiger fact"

> "Alexa or Hey Google, ask endangered species facts how many species are endangered"

> "Alexa or Hey Google, open endangered species facts"

## Test locally

```sh
$ jovo run
```

## Build

```sh
$ jovo build
```

## Deploy

```sh
$ jovo deploy
```
